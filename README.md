# TP Node.js - Gestión de Productos

Este proyecto permite gestionar productos usando la API de FakeStore desde la terminal.

## Comandos

- **Listar productos:**  
  `npm run start GET products`

- **Obtener producto por ID:**  
  `npm run start GET products/<id>`

- **Crear producto:**  
  `npm run start POST products <title> <price> <category>`

- **Eliminar producto:**  
  `npm run start DELETE products/<id>`

## Tecnologías
- Node.js (ESModules)
- Fetch API
